## trouble shooting 

* 지정한 장치 경로 또는 파일에 액세스할 수 없습니다

  V3 백신 관련 이슈일 확률이 높음

  V3 설정 > PC 실시간 검사 > 클라우드 평판 기반 실행 차단 사용 **해제**
